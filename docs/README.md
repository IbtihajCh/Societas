# Documentation

Canonical project documentation, architecture decisions, and development guides.

## Key Files

- `adr/` — Architecture Decision Records (ADRs)
- `guides/` — Setup, coding standards, branching strategy, AI agent rules
- `references/` — Architecture overview, glossary
- `templates/` — Feature specs, meeting notes

## How to Use

Browse locally or open the Obsidian vault at `vault/` for linked navigation.

## Dependencies

- None (Markdown)
