# Meetings

All meeting notes organized by date.

## Structure

```
040-Meetings/
├── 2026-07-07-kickoff.md
├── 2026-07-08-architecture-review.md
└── standups/
    └── 2026-07-07.md
```

## Templates

Use the [meeting note template](../../docs/templates/meeting-notes.md) for consistency.

## Conventions

- Daily standups: `standups/YYYY-MM-DD.md` with 3-bullet format (done, doing, blockers)
- Tech reviews: `YYYY-MM-DD-topic.md` with decision outcomes linked to ADRs
- Retrospectives: stored in `030-Sprints/Sprint-NNN/retro.md`
